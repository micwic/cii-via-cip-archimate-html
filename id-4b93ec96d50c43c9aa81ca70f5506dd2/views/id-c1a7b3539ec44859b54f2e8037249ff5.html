3.2.3.1 CII via CIP - Scenario fully automated with BPA - Application Usage ()
In this scenario, the B2C portal only have a link that open a conversion space in a given HTML DOM node element that generates a case form initialisation for customer service request case with the necessary relevant information for a full automated execution of the end-to-end process. It is important to note that the functional scope is fully achieved by the case model, in a no-code/low-code model-driven approach - which drastically limits the need for coordination between the portal development teams and the specialized business analysts who support case management.
