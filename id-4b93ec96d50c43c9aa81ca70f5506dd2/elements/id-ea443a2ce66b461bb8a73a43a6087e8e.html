inetOrgPerson ()
The inetOrgPerson object class extends organizationalPerson and is commonly used to represent individual user accounts in an LDAP directory. It includes standard attributes like cn (commonName), sn (surname), mail (email address), and uid (user identifier), making it suitable for managing people in corporate directories, authentication systems, and user provisioning.
