Distinguished Name (DN) ()
A Distinguished Name (DN) is a unique, fully qualified identifier in an LDAP directory that defines the exact location of an entry within the directory's hierarchical structure. It is constructed by concatenating all the relative distinguished names (RDNs) from the entry up to the root, ensuring each entry is referenced unambiguously. In the current diagram, the DN represents the LDAP node along with its associated content.
