person ()
The person object class is a fundamental structural class in LDAP directories, representing an individual. It includes essential attributes such as cn (commonName) and sn (surname). As a base class, it is often extended by other object classes like organizationalPerson and inetOrgPerson to add more specific attributes for organizational and user management.
