Activate CII via CIP ()
Activate CII via CIP on the smart meter. This technical task is performed on the head and system. The head end system generates a key and configures the smart meter using remote maintenance services. These activities are generally carried out at times that do not interfere with other operations on the intelligent measuring system. Communications between the HES and smart meters can be more or less instantaneous, depending on the communication system and protocols in place. In case of an unrecoverable error the smart meter is reset to previous configuration and the data are rollbacked except tzhe rejection/termination reason.
