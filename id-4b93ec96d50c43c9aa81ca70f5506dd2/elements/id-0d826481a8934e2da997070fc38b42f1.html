FlowableFlowApp ()
The FlowableFlowApp component is designed as a high-level React component that renders a custom view for a Flowable flow application. It functions by first extracting custom view properties—such as flags to show the user profile or login, and whether to disable routing—from a dedicated hook. These properties are then passed to a container component, which manages the overall layout and user interface settings. Within this container, the component further delegates rendering to a subcomponent that handles routing-specific logic. This subcomponent uses URL query parameters to determine the current flow application identifier (flowAppId). It employs a memoization hook to optimize performance by recalculating the identifier only when relevant values change. The flowAppId is then provided via a context provider, ensuring that the internal components, including the main application logic encapsulated in FlowableFlowAppInternal, have access to the correct contextual information. Overall, the component is architected to separate concerns: one part deals with the custom view’s layout and display settings, while the other focuses on routing and context configuration for the application’s internal logic.
