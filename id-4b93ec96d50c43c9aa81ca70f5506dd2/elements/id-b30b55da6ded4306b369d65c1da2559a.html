Flowable Work Components Flowable Work is a commercial platform designed for dynamic and adaptive process and case management. It allows organizations to model and orchestrate business applications through processes, decision rules, case structures, and dynamic forms. Built on the robust open-source Flowable Engine, the platform supports open standards such as BPMN, CMMN, and DMN, and aims to increase business agility through continuous improvement of operational workflows. Below are the core components deployed in a Flowable Work environment: Flowable Work Engine The central execution engine responsible for running BPMN-based processes, CMMN-based case models, and DMN-based decision logic. It provides a flexible and extensible runtime platform supporting standard-based automation. Flowable Task A rich user interface for managing tasks, cases, and workflows. It allows end-users to track, execute, and collaborate on assigned tasks within the business context. Flowable REST API A comprehensive interface enabling external systems and applications to interact programmatically with Flowable Work. It exposes services for managing processes, cases, decisions, tasks, users, and administrative functions. Flowable Forms A toolset for creating and managing adaptive user interface forms integrated into processes and cases. It supports conditional logic, data binding, and visual configuration to suit process-driven UIs. In runtime, Flowable Work Engine, Flowable Task, and Flowable Forms, together with their associated REST APIs, are deployed and actively support business execution. Flowable Ecosystem These additional components extend the development, administration, and identity management capabilities of the Flowable platform: Flowable Design A cloud-based visual modeling environment for designing BPMN, CMMN, and DMN models. It enables business analysts and developers to collaboratively design, test, and deploy models from a browser. Flowable Admin A web-based administration dashboard for monitoring, diagnosing, and managing the execution environment. It includes tools for runtime oversight, user and group management, and system diagnostics. Flowable IDM (Identity Management) Provides centralized identity and access control. It supports the configuration of users, roles, and groups across Flowable applications, either standalone or as an integration point with external identity providers (e.g., LDAP, Keycloak).
