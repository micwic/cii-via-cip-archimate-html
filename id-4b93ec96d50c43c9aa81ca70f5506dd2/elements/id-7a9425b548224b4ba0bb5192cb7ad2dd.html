Docker Container & Ephemeral storage ()
Docker Container (Generic Node) represents a standardized container instance created and executed by Docker Desktop. It serves as a generalized abstraction of project-specific containers defined in docker-compose.yml for the CII-via-CIP project. Each container node is associated with a Device representing its internal IP address, dynamically allocated by Docker Desktop's DHCP service. The system interfaces within the container define the ports and protocols used for communication. This generic node ensures a consistent representation across different views while maintaining flexibility in network configuration.
