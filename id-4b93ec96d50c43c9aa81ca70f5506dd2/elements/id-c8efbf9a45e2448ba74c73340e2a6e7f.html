groupOfUniqueNames ()
The groupOfUniqueNames object class in LDAP is used to define groups whose members are uniquely identified by their Distinguished Names (DNs). It is commonly employed to represent security groups, distribution lists, or other collections of directory entries, ensuring that each member is distinct. Typically, entries of this class include one or more uniqueMember attributes that list the DNs of the group members.
