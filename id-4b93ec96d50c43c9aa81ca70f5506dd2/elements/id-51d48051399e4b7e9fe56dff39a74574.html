organizationalUnit ()
The organizationalUnit object class represents a subdivision within an organization, such as a department or business unit. It is a structural object class that typically includes the ou (organizationalUnitName) attribute to store the unit's name. It is commonly used in LDAP directories to organize entries hierarchically within an organization or domain.
