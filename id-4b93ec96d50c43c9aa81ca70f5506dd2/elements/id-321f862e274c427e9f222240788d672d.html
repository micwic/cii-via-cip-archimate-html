Neo4j Core Database Service ()
Fundamental graph database service providing core ACID transaction management, native graph storage, and Cypher query execution engine. Handles node and relationship persistence, transaction logging, constraint enforcement, and index management. Serves as the foundational data layer for all specialized search and analytical services. Manages database lifecycle, schema operations, and provides essential graph database primitives for higher-level services.
