Scenario 2b - CII via CIP - BPA fully automated with RPA ()
A fully automated approach using RPA to bypass the time and cost associated with traditional integration, utilizing existing systems user interfaces instead of system interfaces. As you can see, in this scenario, the business orchestration is fully automated through RPA by using application client desktop or web interfaces. No human interaction with internal stakeholders is required, except for exception handling, which is possible but not modeled at this level (happy path abstraction). 'Serving' and 'Realization' relationships are not duplicated in the model to avoid visual redundancy.
