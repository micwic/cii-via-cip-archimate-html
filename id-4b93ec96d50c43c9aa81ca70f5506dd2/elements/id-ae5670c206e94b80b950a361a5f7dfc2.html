mailhog/mailhog ()
This Docker image provides MailHog, a developer-focused email testing tool that emulates an SMTP server and captures outgoing emails for inspection. It includes a web-based user interface and a REST API for viewing and managing messages, making it ideal for local development and automated testing scenarios.
